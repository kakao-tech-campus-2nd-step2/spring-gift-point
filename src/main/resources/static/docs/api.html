회원가입

기본정보

Method URL Authentication method

POST

/api/members/register

NONE

설명: 새로운 사용자가 회원가입을 위해 이메일과 비밀번호를 제출하여 계정을 생성합니다. 성공적으로 가입하면 액세스 토큰을 받게 됩니다.

요청

Request fields

Path Type Description Required

email

String

사용할 이메일

O

password

String

사용할 비밀번호

O

Request body

{
  "email" : "test@test.com",
  "password" : "1234"
}

응답

Response fields

Path Type Description

token

String

액세스 토큰

Response body

{
  "token" : "{access_token}"
}

예제

HTTP request

POST /api/members/register HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 54
Host: localhost:8080

{
  "email" : "test@test.com",
  "password" : "1234"
}

HTTP response

HTTP/1.1 200 OK
Vary: Origin
Vary: Access-Control-Request-Method
Vary: Access-Control-Request-Headers
Content-Type: application/json
Content-Length: 32

  "token" : "{access_token}"
}

카카오 로그인

/api/kakaologin

NONE

설명: 기존 사용자가 카카오 로그인을 시도합니다. 성공적으로 로그인하면 액세스 토큰을 받게 됩니다.

}

Host: localhost:8080
