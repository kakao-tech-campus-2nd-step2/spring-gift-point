# 6주차 과제
## step 3
### 기능 요구 사항
상품 구매에 사용할 수 있는 포인트 기능을 구현한다.
    포인트는 사용자별로 보유한다.
    포인트 차감 방법 등 나머지 기능에 대해서는 팀과 논의하여 정책을 결정하고 구현한다.
    e.g. 총 금액의 5%가 할인된다.
    API 명세는 팀과 협의하여 결정하고 구현한다.
    관리자 화면에서 포인트를 충전할 수 있다.

### 구현 기능
    - [O] order 기능에 포인트 사용 기능 
    - [O] 주문 완료 후 포인 트 적립 기능
    - [O] 사용자 포인트 조회 기능
